# Meesho Hyperlocal Demand Forecasting & Inventory Optimization

## Project Overview

This project analyzes hyperlocal demand patterns across multiple cities and product categories. The objective is to forecast future demand and provide inventory optimization recommendations to reduce stock shortages and improve operational efficiency.

## Tools Used

* SQL Server
* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Power BI

## Project Workflow

1. Data Cleaning and Preparation
2. SQL-Based Business Analysis
3. Exploratory Data Analysis (EDA)
4. Demand Forecasting Model
5. Interactive Power BI Dashboard
6. Inventory Recommendations

## Key Insights

* High-demand cities were identified using order and revenue analysis.
* Category-wise demand trends were analyzed.
* Forecasting was used to estimate future demand.
* Inventory recommendations help reduce stockout situations.

## Dashboard Features

* KPI Cards

* City-wise Demand Distribution
